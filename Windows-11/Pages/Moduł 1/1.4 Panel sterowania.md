# 1.5 Panel sterowania

## Wprowadzenie

<img src="https://raw.githubusercontent.com/Edu-Koala-V/images/main/Windows_11/Images/panel-sterowania-icon.png" alt="Control panel Icon" style="display: inline-block; height: 1.2rem; vertical-align: middle;">*Panel sterowania* to narzędzie w _**Windows 11**_ do zmiany ustawień systemu – od sieci, przez zabezpieczenia, po zarządzanie urządzeniami.

**_Microsoft_** od _**Windows 10**_ dąży do zastąpienia go aplikacją <img src="https://raw.githubusercontent.com/Edu-Koala-V/images/main/Windows_11/Images/windows-settings-icon.png" alt="Windows Settings Icon" style="display: inline-block; height: 1.2rem; vertical-align: middle;">*Ustawienia*, i z każdą aktualizacją (np. `22H2`, `23H2`, `24H2`) więcej opcji migruje do **Ustawień**.

Wiele ustawień wciąż jest jednak dostępnych tylko w **Panelu sterowania**, a niektóre opcje w **Ustawieniach** przekierowują do **Panelu sterowania** przez linki.

## Uruchamianie Panelu sterowania 

- **Wyszukiwarka**: <span style="white-space: nowrap;"><kbd class="Win"></kbd> + <kbd>S</kbd></span>, wpisz „*Panel sterowania*”.
- **Uruchom**: <span style="white-space: nowrap;"><kbd class="Win"></kbd> + <kbd>R</kbd></span>, wpisz `control` i naciśnij <kbd>Enter</kbd>.

**Widok** możesz zmienić w prawym górnym rogu (menu „**Widok**”).
- **Kategorie**: Ustawienia pogrupowane (np. **System i zabezpieczenia**).
- **Duże/małe ikony**: Lista wszystkich opcji (np. **Opcje zasilania**).

![Panel sterowania – widok „Duże ikony”](https://raw.githubusercontent.com/Edu-Koala-V/images/main/Windows_11/Images/Panel-sterowania.webp)

## Funkcje Panelu sterowania

**Panel sterowania** oferuje dostęp do ustawień systemowych, które częściowo pokrywają się z aplikacją **Ustawienia**, ale wiele opcji jest unikalnych lub bardziej szczegółowych. Poniżej podział na funkcje dostępne bezpośrednio w **Panelu sterowania**, te przekierowujące do **Ustawień**, uruchamiające oddzielne okna oraz przystawki Windows.

### 1. Opcje dostępne bezpośrednio w Panelu sterowania

Te ustawienia uruchomisz bezpośrednio w **Panelu sterowania**, bez przekierowań do aplikacji **Ustawienia**:

| Opcja | Opis |
|-------|------|
| **Autoodtwarzanie** | Konfiguracja akcji dla nośników (np. pendrive, karta SD). |
| **Konta użytkowników** | Zmiana typu konta (Standardowe/Administrator). |
| **Miejsce do magazynowania** | Konfiguracja RAID 0 (łączenie dysków) lub RAID 1 (duplikacja danych). |
| **Szyfrowanie dysków funkcją BitLocker** | Szyfrowanie dysku/partycji z hasłem lub TPM.|
| **Centrum sieci i udostępniania** | Ręczna konfiguracja IP, zmiana nazwy interfejsu sieciowego. |
| <span style="white-space: nowrap;">**Kopia zapasowa i przywracanie (Windows 7)**</span> | Tworzenie pełnych kopii zapasowych, przywracanie plików. |
| **Programy i funkcje** | Odinstalowywanie programów, włączanie lub wyłączanie funkcji systemu Windows. |
| **Centrum ułatwień dostępu** | Funkcje dla osób z niepełnosprawnościami (np. lupa). |
| **Historia plików** | Archiwizacja plików z katalogu użytkownika (`C:\Users\Nazwa`) z kontrolą wersji. |
| **Menedżer poświadczeń** | Zarządzanie hasłami do sieci, aplikacji. |
| **Odzyskiwanie** | Tworzenie punktów przywracania, przywracanie systemu. |
| **Zabezpieczenia i konserwacja** | Usuwanie zbędnych plików, naprawa błędów systemowych. |
| **Połączenia RemoteApp i pulpitu** | Konfiguracja zdalnych pulpitów i aplikacji. |



### 2. Opcje uruchamiające aplikację Ustawienia

Te opcje w **Panelu sterowania** przekierowują do **Ustawień**:

| Opcja | Gdzie otwiera |
|-------|--------------|
| **Programy domyślne** | **Ustawienia** → **Aplikacje** → **Aplikacje domyślne** |
| **Urządzenia i drukarki** | **Ustawienia** → **Urządzenia** → **Drukarki i skanery** |
| **Pasek zadań i nawigacja** | **Ustawienia** → **Personalizacja** → **Pasek zadań** |
| **Rozwiązywanie problemów** | **Ustawienia** → **System** → **Rozwiązywanie problemów** |
| **Czcionki** | **Ustawienia** → **Personalizacja** → **Czcionki** |
| **System** | **Ustawienia** → **System** → **Informacje** |

### 3. Opcje uruchamiające dedykowane okna

Te ustawienia otwierają dedykowane okna konfiguracyjne:

| Opcja | Opis |
|-------|------|
| **Data i godzina** | Konfiguracja czasu, strefy czasowej. |
| **Opcje indeksowania** | Zarządzanie indeksem wyszukiwania plików. |
| **Zarządzanie kolorami** | Kalibracja kolorów monitora. |
| **Dźwięk** | Konfiguracja głośników, mikrofonu. |
| **Mysz** | Ustawienia kursora, prędkości. |
| **Opcje internetowe** | Konfiguracja przeglądarki **Edge** (np. proxy). Wcześniej dotyczyło **Internet Explorer**. |
| **Telefon i modem** | Ustawienia modemów (rzadko używane, archaiczne). |
| **Mail (Microsoft Outlook)** | Konfiguracja profili **Outlook**. |
| **Region** | Format daty, waluty. |
| **Klawiatura** | Ustawienia prędkości pisania, układów. |
| **Opcje Eksploratora plików** | To samo okno co w opcjach <img src="https://raw.githubusercontent.com/Edu-Koala-V/images/main/Windows_11/Images/file-explorer-icon.png" alt="File Explorer Icon" style="display: inline-block; height: 1.2rem; vertical-align: middle;">*Eksploratora plików*. |

### 4. Przystawki i narzędzia Windows

Niektóre opcje uruchamiają przystawki **MMC** lub katalogi:
- **Menedżer urządzeń**: Otwiera okno z listą urządzeń (np. do aktualizacji sterowników).
- **Narzędzia systemu Windows**: Otwiera folder w **Eksploratorze plików**: <span style="white-space: nowrap;">`C:\ProgramData\Microsoft\Windows\Start Menu\Programs\Administrative Tools`.</span>  
  W ciemnym motywie folder jest zbugowany:

## BitLocker

- **Ścieżka**: <kbd class="win-menu-btn">System i zabezpieczenia</kbd> → <kbd class="win-menu-btn">Szyfrowanie dysków funkcją BitLocker</kbd>.
- **Opis**: Szyfruje dysk lub partycję, zabezpieczając dane przed nieautoryzowanym dostępem. Używa algorytmu `AES` ($$128$$ lub $$256$$ bitów) i wymaga hasła, klucza USB lub modułu **TPM** (*Trusted Platform Module*).  
Po zaszyfrowaniu nieodblokowana partycja pokazuje tylko ikonę z zamkniętą kłódką – nie da się sprawdzić pojemności ani zajętego miejsca (np. przez właściwości lub <span style="white-space: nowrap;"><kbd>Alt</kbd> + <kbd>Enter</kbd></span>).  
Bez klucza/hasła dane są niedostępne, nawet po przełożeniu dysku do innego komputera.

![Zabezpieczony BitLockerem pendrive](https://raw.githubusercontent.com/Edu-Koala-V/images/main/Windows_11/Images/BitLocker-zabezpieczony-dysk.webp)

- **BitLocker To Go**: Wersja dla nośników wymiennych (np. pendrive’ów). Działa podobnie, ale nie wymaga TPM – wystarczy hasło lub klucz odzyskiwania. Zaszyfrowany nośnik działa na innych komputerach z Windows (po podaniu hasła).

<div data-hint="info">

**Zastosowanie**: Zalecane dla urządzeń z wrażliwymi danymi, np. firmowych laptopów, które mogą zostać skradzione. Samo hasło do systemu nie chroni danych, bo dysk wyjęty z komputera można zamontować w innym i uzyskać pełen dostęp do danych.

</div>

<div data-hint="warning">

**Wpływ na wydajność**: Szyfrowanie obciąża procesor (zwłaszcza bez wsparcia od modułu **TPM**), spowalniając odczyt/zapis.

</div>

- **Kroki konfiguracji**:
  1. Wybierz dysk/partycję (np. `D:\`), kliknij <kbd class="link-os-windows">Włącz funkcję BitLocker</kbd>.
  2. Wybierz metodę: hasło (minimum $$8$$ znaków, używaj kombinacji liter, cyfr, znaków specjalnych) lub klucz USB.
  3. Zapisz klucz odzyskiwania w bezpiecznym miejscu (np. osobny dysk, wydruk).
  4. Wybierz zakres: cały dysk (bezpieczniejsze) lub zajęte miejsce (szybsze).
  5. Uruchom szyfrowanie i czekaj (czas zależy od rozmiaru i typu urządzenia pamięci masowej).

<div data-hint="danger">

**UWAGA**: Dysk musi być w formacie **`NTFS`**.

</div>

![BitLocker w Panelu sterowania](https://raw.githubusercontent.com/Edu-Koala-V/images/main/Windows_11/Images/BitLocker-panel-sterowania.webp)

## Historia plików

- **Ścieżka**: <kbd class="win-menu-btn">System i zabezpieczenia</kbd> → <kbd class="win-menu-btn">Historia plików</kbd>.
- **Opis**: Automatycznie archiwizuje pliki z katalogu `C:\Users\[NazwaUżytkownika]\` (np. **Pulpit**, **Dokumenty**, **Obrazy**) na dodatkowy dysk (zewnętrzny, wewnętrzny lub sieciowy) z kontrolą wersji. Tworzy kopie zmienionych plików w określonych odstępach czasowych, umożliwiając przywrócenie wcześniejszych wersji.
  <div data-hint="warning">

  Funkcja działa tylko dla użytkownika, który ją włączył, a nie dla całego systemu.

  </div>
  <div data-hint="info">
  
  *Kontrola wersji* to proces zarządzania zmianami w plikach, dokumentach, kodzie źródłowym lub innych zasobach cyfrowych. Umożliwia śledzenie zmian oraz ich cofanie w razie potrzeby.
  
  **Historia plików** w systemie Windows to uproszczona forma kontroli wersji. Automatycznie, w określonych odstępach czasu, tworzy kopie zmodyfikowanych plików w całości. Działa na podstawie metadanych – porównuje datę modyfikacji pliku z wersjami już zapisanymi i w razie wykrycia zmian zapisuje nową kopię.
  
  </div>

Kopie plików mają zmodyfikowaną nazwę, do której dodawana jest metadana z datą ostatniej modyfikacji.

![Przykład działania Historii plików](https://raw.githubusercontent.com/Edu-Koala-V/images/main/Windows_11/Images/pliki-historia-plików.webp)

<kbd class="link-os-windows">Ustawienia zaawansowane</kbd> oferują następujące możliwości:
- **Częstotliwość kopiowania**: Minimum co $$10$$ minut, maksimum codziennie (domyślnie co $$1$$ godzina).
- **Czas przechowywania**: Zawsze (domyślnie), dopóki nie zabraknie miejsca, lub od $$1$$ miesiąca do $$2$$ lat.

**Katalogi użytkownika podlegające archiwizacji**:

![Katalogi użytkownika w Historii plików](https://raw.githubusercontent.com/Edu-Koala-V/images/main/Windows_11/Images/Historia-plików-katalogi-kopiowane.webp)

<div data-hint="danger">

**Uwaga**: Ręczne usunięcie folderu `FileHistory` na dysku docelowym sprawi, że próba uruchomienia archiwizacji nie powiedzie się.

![Błąd po usunięciu folderu FileHistory](https://raw.githubusercontent.com/Edu-Koala-V/images/main/Windows_11/Images/delete-fileHistory.webp)

</div>

## Opcje zasilania

- **Ścieżka**: <kbd class="win-menu-btn">Sprzęt i dźwięk</kbd> → <kbd class="win-menu-btn">Opcje zasilania</kbd>.
- **Opis**: Umożliwia konfigurację planów zasilania (np. **Zrównoważony**, **Wysoka wydajność**, **Oszczędzanie energii**), czasu wyłączania ekranu, dysku twardego (dotyczy **HDD** i **SSD**), uśpienia, hibernacji, zachowania przycisku zasilania i zamykania klapy laptopa.  
  W aplikacji <span style="white-space: nowrap;"><img src="https://raw.githubusercontent.com/Edu-Koala-V/images/main/Windows_11/Images/windows-settings-icon.png" alt="Windows Settings Icon" style="display: inline-block; height: 1.2rem; vertical-align: middle;">*Ustawienia*</span> dostępne są podobne opcje (np. czas uśpienia, hibernacji), ale <img src="https://raw.githubusercontent.com/Edu-Koala-V/images/main/Windows_11/Images/panel-sterowania-icon.png" alt="Control panel Icon" style="display: inline-block; height: 1.2rem; vertical-align: middle;">*Panel sterowania* oferuje bardziej szczegółową kontrolę, w tym zaawansowane ustawienia (np. minimalna/maksymalna wydajność procesora).


![Opcje zasilania w Panelu sterowania](https://raw.githubusercontent.com/Edu-Koala-V/images/main/Windows_11/Images/panel-sterowania-opcje-zasilania.webp)
